Cyclistic bike sharing analysis-Case Study

This is a case study completed as part of the Google Data Analytics Certification Project, which analyzes how annual members and casual riders use Cyclistic bikes differently. The analysis is based on data collected over the previous 12 months. As part of the project, various skills related to R were utilized to perform the analysis. These skills included data cleaning, data transformation, exploratory data analysis, and data visualization. Specifically, the following steps were taken:

  • Data Cleaning and Transformation: The raw data was cleaned and transformed to ensure that it was in a usable format for analysis. This involved tasks such as removing duplicates, handling missing values, and creating new variables based on existing data.

  • Exploratory Data Analysis: The cleaned and transformed data was then explored using various statistical techniques such as summary statistics, correlation analysis, and data visualization. This allowed for a better understanding of the data and helped to identify any patterns or trends that could be of interest.

  • Data Visualization: Finally, the results of the analysis were presented using various data visualization techniques such as bar charts, scatterplots, and heatmaps. These visualizations helped to communicate the key findings of the analysis in a clear and concise manner. Overall, this project provided an opportunity to utilize a variety of skills related to R and data analysis, including data cleaning and transformation, exploratory data analysis, and data visualization. The insights gained from the analysis have the potential to inform future marketing and operational strategies for Cyclistic, and demonstrate the value of data-driven decision making.
