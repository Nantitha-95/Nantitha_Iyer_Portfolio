Data Cleaning using Excel(Covid Cases & Deaths Worldwide)

This project focuses on performing data cleaning tasks using the Excel tool to analyze global COVID-19 cases and deaths. The dataset used for analysis was obtained from Kaggle and includes columns such as Serial Number,Country,Total Cases,Total Deaths,Total Recovered, Active Cases,Total Test, and Populataion. The following steps were undertaken to ensure data accuracy and consistency:

  • Renaming column headers: The column headers were renamed to provide a clear and understandable description of the data they represent. This helps in improving the overall comprehension of the dataset.

  • Data Type Transformation: The data types of columns were reviewed and transformed as needed. Numeric columns were converted to numeric format, date/time columns were converted to the appropriate format, and categorical columns were ensured to be recognized as text or categorical data. This step ensures that the data is stored in the correct format for accurate analysis and calculations.

  • Removing duplicates: Duplicate entries can lead to inaccurate analysis and skewed results. To ensure data accuracy, duplicate entries were identified and removed from the dataset. This step helps in maintaining data integrity and avoiding duplicate counting.

  • Checking for missing data: Missing data can create gaps in the dataset and affect the analysis. To address this, the dataset was thoroughly checked for missing values. Whenever missing values were found, appropriate actions were taken to replace them with suitable values.

  • Checking for spelling mistakes: Spelling mistakes in the dataset can cause discrepancies and hinder data analysis. A careful review of the dataset was conducted to identify and correct any spelling mistakes. This ensures consistent and accurate data across the dataset.

  • Checking for errors in the entries for total cases and total deaths: The total cases and total deaths entries were scrutinized for potential errors using a formula. The formula (=D2>C2) compares the value in cell D2 (representing total deaths) with the value in cell C2 (representing total cases). If the total deaths value is greater than the total cases value, it indicates a potential error. This check helps identify inconsistencies and outliers in the data.
