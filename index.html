Data Analysis using SQL(World University Rankings)

This project focuses on analyzing world university rankings using SQL and visualizing the findings using Tableau Public. The dataset used for analysis was obtained from Kaggle and includes columns such as World rank, Institutions, National rank, Education rank, Employability rank, Faculty rank, Research rank, and Score.As part of this project, various skills related to SQL were utilized to perform the analysis. These skills included:

  • Data Cleaning and Transformation: The raw data was cleaned and transformed to ensure that it was in a usable format for analysis. This involved tasks such as removing duplicates, handling missing values, and creating new variables based on existing data.

  • Data aggregation and grouping: The data was aggregated and grouped to obtain summary statistics such as number of universities by country, and top universities.

  • Data filtering and querying: Data was filtered and queried to answer specific research questions such as which countries have the most universities in the top 100 rankings and which univerity tops based on various criteria.

Overall, this project provided an opportunity to utilize a variety of skills related to SQL and data analysis. The insights gained from the analysis have the potential to inform future decisions related to university rankings and demonstrate the value of data-driven decision making.

Data Analysis using SQL(Youtube Trending Videos Analysis)

This project analyzed a dataset of trending YouTube videos using SQL. The dataset contains information on videos that trended in various countries, including their video_id, Country, publishedat, channelTitle, categoryid, trending_date, view_count, likes, dislikes, comment_count, id, and title of the category.As part of this project, various skills related to SQL were utilized to perform the analysis. These skills included:

  • Data Cleaning and Transformation: The raw data was cleaned and transformed to ensure that it was in a usable format for analysis. This involved tasks such as removing duplicates, handling missing values, and creating new variables based on existing data.

  • Data aggregation and grouping: The data was aggregated and grouped to obtain summary statistics such as average views, likes, and comments by category and country.

  • Joining and merging datasets: Multiple datasets were merged and joined together to create a more comprehensive analysis. This involved tasks such as joining video data with category data to obtain the category of each video.

  • Data filtering and querying: Data was filtered and queried to answer specific research questions such as which categories of videos tend to get the most views and which countries tend to have the most popular videos.

Overall, this project provided an opportunity to utilize a variety of skills related to SQL and data analysis. The insights gained from the analysis have the potential to inform content creators and marketers on how to optimize their videos for maximum engagement and views, and demonstrate the value of data-driven decision
